## 🚀 Getting Started

### Prerequisites

- Modern web browser
- Git (for cloning)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/johndhussey/test-weather-widget-v2.git
cd test-weather-widget-v2
```

2. Open locally:
```bash
# Option 1: Open directly
open index.html

# Option 2: Local server (recommended)
python3 -m http.server 8000
# Then visit: http://localhost:8000
```

## 📖 Usage

The application is ready to use out of the box. Simply open the live demo link above or run locally using the installation steps.
